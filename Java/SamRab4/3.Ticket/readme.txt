Ticket класс:

Класс представляет сущность "Билет" с аннотацией @Entity, что указывает на то, что объекты этого класса могут быть сохранены в базе данных.
Имеет уникальный идентификатор (id), который генерируется автоматически с использованием стратегии GenerationType.AUTO.
Аннотация @Table(name = "ticket") указывает на таблицу базы данных, в которой будут храниться объекты класса.
Поля класса:

seat: Номер места в поезде, который может быть любым уникальным идентификатором места.
status: Статус билета, использующий перечисление TicketStatus с аннотацией @Enumerated(EnumType.STRING).
passengerId: Идентификатор пассажира, связанный с забронированным местом.
train поле:

Поле, обозначающее связь с поездом, к которому относится билет. Используется аннотация @ManyToOne, что означает, что у одного билета может быть только один поезд.

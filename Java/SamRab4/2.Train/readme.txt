Train класс:

Класс представляет сущность "Поезд" с аннотацией @Entity, что указывает на то, что объекты этого класса могут быть сохранены в базе данных.
Имеет уникальный идентификатор (id), который генерируется автоматически с использованием стратегии GenerationType.AUTO.
Аннотация @Table(name = "train") указывает на таблицу базы данных, в которой будут храниться объекты класса.
Поля класса:

from, to: Место отправления и место назначения поезда соответственно.
departureTime, arrivalTime: Время отправления и прибытия поезда соответственно. Используется аннотация @Temporal(TemporalType.TIMESTAMP), чтобы указать, что в базе данных хранится дата и время.
tickets поле:

Поле, обозначающее связь с билетами в поезде. Используется аннотация @OneToMany, что означает, что у одного поезда может быть много билетов.
mappedBy = "train" указывает на поле train в классе Ticket, которое устанавливает обратную связь.
addTicket и removeTicket методы:

Методы для удобного добавления и удаления билетов из поезда.
Поддерживают обратное установление связи, чтобы обеспечить целостность данных.

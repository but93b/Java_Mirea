MapperTicketDTO класс:

Отвечает за преобразование объектов типа Ticket в объекты типа TicketDTO.
Метод mapTicketDTO:

Статический метод, который выполняет преобразование объекта Ticket в объект TicketDTO.
Создает новый объект TicketDTO и устанавливает его поля на основе полей объекта Ticket.
Устанавливает идентификатор поезда из объекта Train, связанного с билетом, если он присутствует.
Возвращает созданный объект TicketDTO.

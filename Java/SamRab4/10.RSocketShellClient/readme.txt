RSocketShellClient класс:

Класс представляет RSocket клиент, обрабатывающий команды из командной строки.
Конструктор:

Принимает RSocketRequester.Builder и RSocketStrategies для создания клиента.
Использует SocketAcceptor для установки соединения с сервером.
Подключается к серверу RSocket по протоколу TCP с адресом "localhost" и портом 7000.
logout метод:

Вызывается при завершении работы приложения для отключения клиента от сервера.
isConnection метод:

Проверяет наличие соединения с сервером.
book, deleteReservation, trains, tickets, s методы:

Методы, обрабатывающие команды из командной строки для выполнения соответствующих операций с сервером RSocket.
book: Бронирование места в поезде.
deleteReservation: Отмена бронирования места в поезде.
trains: Получение списка всех поездов.
tickets: Получение информации о билетах по id поездов.
s: Остановка текущего потока данных.
Логирование:

Используется библиотека Lombok для автоматической генерации логгера с именем log.
Логируются основные события, такие как подключение, отключение клиента, отправка запросов и т.д.

TicketDTO класс:

Представляет объект передачи данных (DTO) для информации о билете.
Использует аннотацию Lombok @Data для автоматической генерации геттеров, сеттеров, методов toString, equals и hashCode.
Поля класса:

id: Идентификатор билета.
seat: Номер места.
status: Статус билета (забронирован, свободен и т.д.).
passengerId: Идентификатор пассажира.
trainId: Идентификатор поезда.

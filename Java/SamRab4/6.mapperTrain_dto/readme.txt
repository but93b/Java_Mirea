MapperTrainDTO класс:

Отвечает за преобразование объектов типа Train в объекты типа TrainDTO.
Использует SimpleDateFormat для форматирования времени.
Метод mapTrainDTO:

Статический метод, который выполняет преобразование объекта Train в объект TrainDTO.
Создает новый объект TrainDTO и устанавливает его поля на основе полей объекта Train.
Форматирует временные значения, если они присутствуют.
Возвращает созданный объект TrainDTO.

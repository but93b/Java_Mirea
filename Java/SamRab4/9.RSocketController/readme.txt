RSocketController класс:

Класс контроллера для RSocket, отвечающий за обработку запросов от клиентов и взаимодействие с RSocketService.
Аннотация @Controller указывает, что этот класс является компонентом Spring MVC, обрабатывающим входящие сообщения.
Аннотация @RequiredArgsConstructor автоматически внедряет зависимость через конструктор для полей с аннотацией final.
shutdown метод:

Метод, вызываемый при завершении приложения, который освобождает ресурсы, связанные с клиентами.
connectShellClientAndAskForTelemetry метод:

Метод для установления соединения с клиентом по запросу "shell-client".
Добавляет новых клиентов в список клиентов при подключении и удаляет отключенных клиентов при закрытии канала.
Подтверждает соединение клиента и отправляет информацию о свободной памяти.
book, deleteReservation, getAllTrains, getTickets методы:

Методы обработки RSocket запросов с использованием аннотаций @MessageMapping.
book: Бронирование места в поезде.
deleteReservation: Отмена бронирования места в поезде.
getAllTrains: Получение списка всех поездов.
getTickets: Получение информации о билетах по id поездов.
Логирование:

Используется библиотека Lombok для автоматической генерации логгера с именем log.
Логируются основные события, такие как подключение, отключение клиента, прием запросов и т.д.

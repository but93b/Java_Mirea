TrainDTO класс:

Представляет объект передачи данных (DTO) для информации о поезде.
Использует аннотации Lombok (@Data, @NoArgsConstructor, @AllArgsConstructor) для автоматической генерации геттеров, сеттеров, конструкторов и других методов.
Поля класса:

id: Идентификатор поезда.
to: Пункт назначения.
from: Пункт отправления.
departureFormattedTime: Отформатированное время отправления.
arrivalFormattedTime: Отформатированное время прибытия.
Lombok аннотации:

@Data: Генерирует геттеры, сеттеры, методы toString, equals и hashCode.
@NoArgsConstructor: Генерирует конструктор без параметров.
@AllArgsConstructor: Генерирует конструктор с параметрами для всех полей класса.

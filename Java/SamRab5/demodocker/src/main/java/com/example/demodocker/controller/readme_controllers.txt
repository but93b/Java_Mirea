HomeController представляет собой 
REST-контроллер для обработки запросов связанных с загрузкой и скачиванием файлов.
Каждый метод имеет комментарии, описывающие его назначение и функциональность.



WebController представляет собой Spring MVC контроллер,
который обрабатывает HTTP GET запросы для корневого пути (/).
Метод index() возвращает строку "index", которая предполагается быть
именем представления для шаблонизатора представлений.

uploadFile метод:

Проверяет, не пустое ли имя загружаемого файла.
Создает буферизированный поток для записи файла на сервер.
Записывает байты файла в поток.
В случае успешной загрузки файла, вызывает метод getListofFiles для обновления списка файлов и возвращает его.
getListofFiles метод:

Получает список файлов в директории.
Создает список для хранения имен файлов.
Проверяет, не пустой ли список файлов.
Добавляет имена файлов в список.
Возвращает список имен файлов.
Оба метода также могут выбрасывать исключения в случае ошибок (например, отсутствия файлов в директории или пустого имени файла).
